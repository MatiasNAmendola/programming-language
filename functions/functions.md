# variables
## basics
```python
a = 1
b = true
c = "hello world"

# a = 1
# b = true
# c = "hello world"
```

## explicit types
```python
a Float = 1
b Int   = 1.5

print a, b
# 1.0, 1
```


## proper function definitions
```python
# variables have been functions all along
def a() Int = 2

def square(x Int) Int =
  x * x

print square(a), square(4)
# 4, 16
```
