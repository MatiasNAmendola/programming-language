# functions
## basics
```python
a = 1
b = True
c = "hello world"

print a, b, c
# 1, True, "hello world"
```


## explicit return types
```python
a Float = 1
b Int   = 1.5

print a, b
# 1.0, 1
```


## complete function definitions
```python
def a() Int = 2

def square(x Int) Int =
  x * x

print square(a), square(4)
# 4, 16
```
